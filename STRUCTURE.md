## Runtime/API
- `backend/` - PHP backend endpoints and services

## Assets
- `css/` - stylesheets
- `js/` - frontend scripts
- `img/` - active image assets
- `img/unused/` - currently unused image files kept for safekeeping
- `partials/` - reusable HTML partials (footer)
